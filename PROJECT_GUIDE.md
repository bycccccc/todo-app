# 项目演示指南

## 📋 项目概述

这是一个完整的任务管理应用（Todo List），专为前端实习面试准备设计。项目展示了现代前端开发的核心技能，包括：

- **HTML5** 语义化标签使用
- **CSS3** 现代布局（Flexbox/Grid）和动画
- **JavaScript** ES6+ 面向对象编程
- **本地存储**（localStorage）持久化数据
- **响应式设计** 移动端适配
- **UI/UX** 用户体验优化

## 🚀 快速开始

1. **打开项目**
   - 直接双击 `index.html` 在浏览器中打开
   - 或在VS Code中安装 **Live Server** 插件，右键点击 `index.html` 选择 "Open with Live Server"

2. **基础操作**
   - 在输入框中输入任务内容，点击"添加任务"或按 **Enter** 键
   - 点击任务前的复选框标记完成/未完成
   - 双击任务项快速切换状态
   - 悬停在任务上显示删除按钮

## 💡 核心功能展示

### 1. 任务管理
- ✅ **添加任务**：输入框 + 添加按钮
- ✅ **删除任务**：悬停显示删除按钮
- ✅ **编辑任务**：点击任务文本直接编辑
- ✅ **状态切换**：复选框/双击切换完成状态

### 2. 数据过滤
- **全部任务**：显示所有任务
- **未完成**：仅显示待办任务
- **已完成**：仅显示已完成任务
- **清除已完成**：批量删除已完成任务

### 3. 数据持久化
- **自动保存**：所有操作自动保存到 localStorage
- **数据导出**：导出为JSON格式，可以复制保存
- **数据导入**：导入外部JSON数据，覆写现有数据
- **数据重置**：清空所有任务数据

### 4. 统计功能
- **总任务数**：当前任务总数
- **待完成数**：等待处理的任务数
- **已完成数**：已完成的任务数

## 🎯 面试演示要点

在面试中展示此项目时，可以重点讲解：

### 技术实现
```javascript
// 1. 面向对象设计 - TodoApp 类
class TodoApp {
    constructor() {
        this.tasks = [];
        this.init();
    }
}

// 2. 事件委托 - 高效的事件处理
this.elements.taskList.addEventListener('click', (e) => {
    if (e.target.classList.contains('task-checkbox')) {
        this.toggleTask(e.target.dataset.id);
    }
});

// 3. 本地存储 - 数据持久化
saveToLocalStorage() {
    localStorage.setItem('todo-tasks', JSON.stringify(this.tasks));
}
```

### 设计亮点
- **响应式布局**：完美适配手机、平板、桌面
- **用户体验**：键盘快捷键、悬停提示、快捷键操作
- **性能优化**：事件委托、批量DOM更新
- **安全性**：HTML转义防止XSS攻击
- **可访问性**：语义化标签、键盘导航支持
