# Todo List - 任务管理应用

一个现代化的任务管理应用，适用于前端实习面试准备项目。

## 功能特性
- ✅ 添加新任务
- ✅ 删除任务  
- ✅ 标记任务完成状态
- ✅ 任务统计（总数、未完成数）
- ✅ 任务过滤（全部/未完成/已完成）
- ✅ 本地存储（刷新页面数据不丢失）
- ✅ 响应式设计（适配手机/平板/桌面）

## 技术栈
- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript (ES6+)

## 项目结构
```
comate-zulu-demo/
├── index.html          # 主页面
├── style.css          # 样式文件
├── script.js          # 逻辑文件
├── README.md          # 项目说明
└── assets/            # 静态资源
    ├── icons/         # 图标资源
    └── screenshots/   # 项目截图
```

## 如何运行
1. 直接打开 `index.html` 在浏览器中运行
2. 或在VS Code中使用Live Server插件
